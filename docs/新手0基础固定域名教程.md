# 🎯 新手0基础固定域名完整教程

## 🚀 目标：让你的网站拥有专属域名（如：yourname.com）

### 📋 准备工作（5分钟）
1. **GitHub账号** - 免费注册 [github.com](https://github.com)
2. **Netlify账号** - 免费注册 [netlify.com](https://www.netlify.com)
3. **域名** - 购买域名（推荐阿里云/腾讯云，约30-60元/年）

---

## 📝 第一步：上传代码到GitHub（10分钟）

### 1. 创建GitHub仓库
1. 登录GitHub → 点击右上角 `+` → `New repository`
2. 填写信息：
   - **Repository name**: `my-website`
   - **Description**: 我的网站
   - 选择 **Public**
   - ✅ 勾选 `Add a README file`
3. 点击 `Create repository`

### 2. 上传你的项目文件
**最简单方法 - 直接上传：**
1. 进入刚创建的仓库
2. 点击 `Add file` → `Upload files`
3. 拖拽你的项目文件到网页上
4. 写提交信息：`初次上传网站文件`
5. 点击 `Commit changes`

✅ **验证**：仓库页面能看到所有文件就成功了！

---

## 🌐 第二步：部署到Netlify获得免费域名（5分钟）

### 1. 连接GitHub
1. 登录Netlify
2. 点击 `Add new site` → `Import an existing project`
3. 选择 `GitHub` → 授权连接

### 2. 部署网站
1. 选择你的GitHub仓库 `my-website`
2. 构建设置：
   - **Build command**: `npm install && npm start`
   - **Publish directory**: `public`
3. 点击 `Deploy site`
4. 等待1-2分钟，获得免费域名：`https://xxx.netlify.app`

✅ **验证**：点击域名能打开你的网站！

---

## 🎯 第三步：绑定你的专属域名（10分钟）

### 1. 购买域名（如果还没有）
**推荐平台：**
- 阿里云域名：[wanwang.aliyun.com](https://wanwang.aliyun.com)
- 腾讯云域名：[dnspod.cloud.tencent.com](https://dnspod.cloud.tencent.com)
- 价格：`.com` 约60元/年，`.cn` 约30元/年

### 2. 配置DNS解析
**在域名管理后台添加记录：**

| 记录类型 | 主机记录 | 记录值 | TTL |
|---------|---------|--------|-----|
| CNAME | @ | your-site.netlify.app | 10分钟 |
| CNAME | www | your-site.netlify.app | 10分钟 |

### 3. Netlify绑定域名
1. 进入Netlify → 你的站点 → `Domain management`
2. 点击 `Add custom domain`
3. 输入你的域名：`yourname.com`
4. 点击 `Verify` → `Yes, add domain`
5. 等待SSL证书自动生成（5-10分钟）

## 🔧 常见问题解决

### ❌ 域名无法访问
**解决步骤：**
1. 检查DNS是否生效：`ping yourname.com`
2. 确认Netlify中域名状态为 `Active`
3. 等待DNS传播（最多24小时）

### ❌ HTTPS证书错误
**解决：**
- Netlify会自动申请Let's Encrypt证书
- 等待10-30分钟自动完成
- 或手动点击 `Renew certificate`

### ❌ 网站样式错乱
**检查：**
- 确认所有文件已上传到GitHub
- 检查 `package.json` 中的构建命令
- 查看Netlify部署日志
