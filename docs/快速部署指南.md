## 📱 方法一：手机也能操作（最简单）

### 1. 准备文件
确保所有文件都在一个文件夹里：
```
你的网站文件夹/
├── index.html
├── css/
├── js/
├── images/
└── 其他文件...
```

### 2. 打开Netlify官网
手机浏览器访问：[https://app.netlify.com](https://app.netlify.com)
- 点击"Sign up"注册（用GitHub/Google账号）
- 或点击"Log in"登录

### 3. 拖拽上传
1. 登录后点击"Add new site"
2. 选择"Deploy manually"
3. **直接拖拽整个文件夹到网页上**
4. 等待上传完成（10-30秒）

### 4. 获得域名
上传完成后立即获得：
`https://xxx.netlify.app`（xxx是随机名称）

### 5. 自定义名称（可选）
点击"Site settings" → "Change site name"
可以改成：`https://mywebsite.netlify.app`

## 🎯 方法三：GitHub同步（适合更新）

### 1. 上传GitHub

## 🔧 常见问题

### ❌ 上传失败
**解决：**
- 检查网络连接
- 确认文件完整
- 重新拖拽上传

### ❌ 页面空白
**解决：**
- 确认有 `index.html` 文件
- 检查文件路径是否正确
- 查看浏览器控制台错误

### ❌ 域名打不开
**解决：**
- 等待1-2分钟部署完成
- 检查是否拼写错误
- 清除浏览器缓存
