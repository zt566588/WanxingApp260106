### 方法2：使用手机热点共享
1. 开启手机热点
2. 让运行服务的设备连接手机热点
3. 获取新的IP地址后使用对应地址访问

### 方法3：使用内网穿透工具（推荐）
如需真正的"随时随地"访问，建议使用内网穿透服务：

#### 1. Ngrok（免费）
```bash
# 安装ngrok
npm install -g ngrok

# 启动穿透
ngrok http 3000
```

#### 2. LocalTunnel（免费）
```bash
# 安装
npm install -g localtunnel

# 启动
lt --port 3000
```

#### 3. 花生壳（国内稳定）
- 官网：https://hsk.oray.com/
- 提供固定域名访问

### 防火墙设置
如果仍然无法访问，请检查：
1. **系统防火墙**: 允许3000端口
2. **路由器设置**: 开启端口转发（如需外网访问）
3. **云服务商安全组**: 开放3000端口（云服务器环境）
